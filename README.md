# Лабораторна робота №6

## Задоєнко Роман ІПЗ-24-4

## Опис проекту
Цей застосунок демонструє інтеграцію **Firebase Authentication** та **Cloud Firestore** у мобільний додаток на базі **React Native** . [cite_start]Основна мета — реалізація надійного механізму автентифікації користувачів та захищеного сховища їхніх персональних даних[cite: 1, 3, 5].

## Реалізований функціонал
[cite_start]У межах лабораторної роботи було впроваджено наступні можливості[cite: 1, 7]:

1.  **Авторизація користувача:**
    * [cite_start]Реєстрація нового акаунта за допомогою Email та пароля[cite: 1, 9].
    * [cite_start]Вхід у систему для існуючих користувачів[cite: 1, 9].
    * [cite_start]Вихід із системи (Sign Out)[cite: 1, 10].
2.  **Збереження персональних даних:**
    * [cite_start]Після входу користувач може заповнити або оновити профіль: **ім'я, вік, місто**[cite: 1, 12].
    * [cite_start]Дані синхронізуються з **Firebase Firestore** у колекції `users`[cite: 1, 13, 25].
    * [cite_start]ID документа відповідає унікальному `uid` користувача[cite: 1, 13].
3.  **Захист доступу:**
    * [cite_start]Навігація розділена на групи `(auth)` та `(app)` за допомогою **Expo Router**[cite: 1, 26].
    * [cite_start]Використано **AuthContext** для централізованого керування станом авторизації[cite: 1, 27].
    * [cite_start]Налаштовано **Firestore Security Rules** для перевірки `uid`, що забороняє доступ до чужих даних[cite: 1, 15, 16].
4.  **Керування обліковим записом:**
    * [cite_start]Редагування полів профілю через форму[cite: 1, 18].
    * [cite_start]**Видалення акаунта** з обов’язковою повторною автентифікацією для безпеки[cite: 1, 19, 20].
5.  **Відновлення паролю:**
    * [cite_start]Можливість зміни або скидання пароля через email за допомогою Firebase API[cite: 1, 21, 22].

## Технічний стек
* **Framework:** React Native (Expo)
* [cite_start]**Navigation:** Expo Router [cite: 1, 26]
* [cite_start]**Backend:** Firebase Auth, Firebase Firestore [cite: 1, 24, 25]
* [cite_start]**State Management:** React Context API (AuthContext) [cite: 1, 27]
